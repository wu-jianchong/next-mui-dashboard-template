# Next.js + MUI Dashboard Template

A starter template for Next.js (App Router) + Material-UI (MUI) + React projects.

## Features

- Sidebar navigation with your menu items
- Clean URLs via route groups
- TypeScript support
- Page routing ready

## Setup

1. Fork this repo
2. `npm install`
3. `npm run dev`
4. Visit http://localhost:3000/dashboard

## Structure

- `app/(features)/`: Routed pages
- `components/layout/`: Sidebar & AppBar
- `theme/`: MUI theme
